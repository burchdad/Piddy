## 🚀 Getting Started

### Quick Start (5 minutes)
```bash
# Install
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Configure
cp .env.example .env
# Edit .env with your credentials

# Run
python -m src.main
# or for Slack integration
bash start-slack.sh
```

### Using in Slack
```
@Piddy Generate a Python FastAPI endpoint for authentication
@Piddy Review this code for security issues
@Piddy Design a database schema for a blog
```

### Using the API
```bash
curl -X POST http://localhost:8000/api/v1/agent/command \
  -H "Content-Type: application/json" \
  -d '{
    "command_type": "code_generation",
    "description": "Generate a REST API",
    "priority": 8
  }'
```

### Using Phase CLIs
```bash
# Phase 5: MLOps
phase5 ml train config.yaml
phase5 observe metrics

# Phase 6: Service Mesh
phase6 mesh configure istio production
phase6 gateway create kong main

# Phase 7: Security
phase7 security scan-dependencies requirements.txt
phase7 backup create-dr-plan services.txt

# Phase 8: AI Operations
phase8 predict forecast api-service 24
phase8 refactor analyze src/app.py python
```
