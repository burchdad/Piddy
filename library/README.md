# Piddy Library

Piddy's built-in knowledge and learning materials.

## Structure

```
library/
├── books/        # Free programming books (48 languages)
├── courses/      # Free online courses & tutorials (51 collections)
├── languages/    # Programming language references & cheatsheets
└── standards/    # Coding standards, best practices, style guides
```

## Usage

Piddy uses this library for:
- Answering programming questions with curated references
- Suggesting learning resources to users
- Following language-specific best practices
- Providing code examples from trusted sources

## Adding Content

Drop `.md`, `.pdf`, or `.txt` files into the appropriate subfolder.
Piddy's knowledge base ingestion will pick them up automatically.
